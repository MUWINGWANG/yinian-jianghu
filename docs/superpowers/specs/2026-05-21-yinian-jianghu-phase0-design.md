## 一、世界观

### 时代背景

**大雍朝 · 天启年间**（架空明清，风格参照明末）

皇权式微，宦官干政，边患频发。朝廷对江湖的管控从「笼络」滑向「利用」，正派门派的政治立场愈发微妙。整体氛围参照《卧虎藏龙》：情感克制、身份枷锁、江湖自由三重张力并存。

### 主角设定

**叶流霜**（玩家可自定义名称）

- **身份**：刑部侍郎之女，精通诗书礼仪，婚事已由家族定下
- **秘密**：自幼偷习武学，内功已有小成（玉娇龙型：高门贵女 × 江湖自我）
- **触发事件**：随父赴任途中，夜宿驿站，隔壁密室交谈——无意听到足以颠覆正派联盟的秘密
- **处境**：无门无派，孤身一人，30 天内须返京完婚；这个秘密烫手，每方都想从她身上取走

### 核心秘密

**正道之蛇**：太虚观掌门玄机子与血煞堂已有十年密约——以竞争对手行动情报换取血煞堂永不攻击太虚观门人。由此促成数起「意外身亡」。叶流霜在驿站偷听到的，正是最新一次交易内容。

## 二、游戏结构

### 30 天日历

| 阶段 | 天数 | 内容 |
|------|------|------|
| 初入江湖 | Day 1–8 | 认识各派人物，建立初始关系 |
| 暗流涌动 | Day 9–20 | 秘密开始扩散，各方压力升级 |
| 一念之间 | Day 21–29 | 关键抉择节点，关系格局定型 |
| 终局 | Day 30 | 根据积累触发 7 种结局之一 |

### 每日结构

```
[ 晨 · 场景叙述 ]  文字 + NPC 立绘，交代当天事件
        ↓
[ 日间 · 核心抉择 ]  1–2 个选项，影响好感度 / 秘密扩散指数
        ↓  （约 1/3 天触发回合制战斗）
[ 傍晚 · 关系波动 ]  简短反馈：某 NPC 态度变化提示
        ↓
[ 夜 · 可选对话 ]  主动拜访 1 位 NPC（消耗「夜访」名额）
                   → 深度对话，推进亲密 / 敬畏等维度
```

### 结局触发条件（待 Phase 5 细化）

7 种结局由以下三轴交叉决定：
1. **秘密归属**：秘密被谁掌握（玄机子 / 血煞堂 / 正义公开 / 主角独吞 / 销毁）
2. **主角身份选择**：返京完婚 / 留在江湖 / 第三条路
3. **核心 NPC 存活与立场**：沈如玉 / 凌霜 / 燕三刀等关键角色的命运

## 三、五大门派

### 正道三派

#### 青云门
- **底色**：儒家
- **定位**：江湖执法者，与朝廷关系暧昧，门规严苛
- **秘密关联**：掌门早知太虚观异动，选择沉默
- **代表武学**：文剑十三式（招式取自书法笔法）
- **标签**：政治嗅觉强 · 剑法刚正 · 内部保守

#### 灵隐宗
- **底色**：佛家
- **定位**：山中古刹，以医术和调解著称，中立调停方
- **秘密关联**：了尘大师是信息守门人，知道的远比表现出来的多
- **代表武学**：金刚护体功 / 慈悲杖法
- **标签**：医毒双绝 · 信息来源广 · 安全港

#### 太虚观 ⚠️
- **底色**：道家
- **定位**：表面超然，暗中经营已久
- **秘密关联**：**核心秘密持有者**，与血煞堂有十年密约
- **代表武学**：太虚剑意 / 无形气劲
- **标签**：内功深厚 · 城府极深 · 消息封锁

### 魔道两派

#### 血煞堂
- **底色**：刺客宗
- **定位**：江湖最恐惧的名字，接受任何委托
- **秘密关联**：密约另一方，与太虚观互为保护伞
- **代表武学**：鬼手七绝 / 百步穿心针
- **标签**：暗器毒术 · 情报网络 · 雇佣性质

#### 荒原帮
- **底色**：绿林草莽
- **定位**：流民、逃兵、被正派驱逐者的聚集地，「魔道」标签是正派强加的
- **秘密关联**：道德最灰色一派，可能成为主角意外盟友
- **代表武学**：荒刀三十六路 / 驼背擒拿手
- **标签**：人多势众 · 野路子功夫 · 反建制

## 五、美术风格基线

### 核心风格

**朱砂红古卷风**：中国古典工笔彩绘插画 × 明代武侠题材 × 电影级叙事调色

### 配色系统

| 色彩 | 色值 | 用途 |
|------|------|------|
| 朱砂红 | `#C0392B` | 主色·强调·印章 |
| 深朱红 | `#8B1A1A` | 悬停·边框 |
| 古纸米 | `#F5E6C8` | 主文字·亮色背景 |
| 旧卷黄 | `#E0CC98` | 次文字 |
| 墨底黑 | `#1A1208` | 主背景 |
| 焦墨褐 | `#2A1E0E` | 面板背景 |
| 赤金 | `#C9A84C` | 高光·门派徽章 |
| 松石青 | `#4A7C59` | 正道门派标识 |

### 字体规范

- **标题**：Noto Serif SC Bold · 22px · 字距 6px
- **章节标题**：Noto Serif SC · 15px · 字距 4px
- **正文叙事**：Noto Serif SC · 12px · 行距 2x
- **印章标签**：serif · 旋转 -8° · 朱砂边框

### 背景纹理

- **亮色模式**（日间/对话）：古卷纸张渐变 `#F5E6C8 → #E0CC98`
- **暗色模式**（夜间/战斗）：墨底暗纹 `#1A1208`，朱砂边框点缀

### 动效规范

| 效果 | 时长 | 缓动 |
|------|------|------|
| 卷轴展开 | 400ms | ease-out |
| 选项浮现 | 逐条 100ms 延迟 | ease-in |
| 印章落下 | 弹性缩放 + 红晕扩散 | spring |
| 好感度变化 | 300ms 数字滚动 | linear |
| 页面切换 | 墨迹化开 500ms | ease-in-out |
| 战斗招式 | 笔触扫过 200ms | ease-out |
